# Inventory Process

1. Capture raw media in `assets/products/raw`.
2. Export optimized WebP assets into `assets/products/optimized`.
3. Record title, category, sport, player, team, year, brand, rarity, pricing, and authentication.
4. Verify PSA, Beckett, SGC, or JSA certificate identifiers.
5. Publish only after pricing, insurance, and sale type are approved.
